# CorvusDossiers

**Threat intelligence reports from the field.**

> Browse reports online: **https://lacrimae0rerum.github.io/CorvusDossiers/**

Public repository of CTI dossiers produced with [CorvusEngine](https://github.com/lacrimae0rerum/CorvusEngine). Each report documents a real threat — phishing kits, smishing campaigns, malicious infrastructure — investigated through passive OSINT only. No active scanning, no form submissions, no exploitation.

## What You'll Find

- Full kill-chain analysis with MITRE ATT&CK mapping
- Infrastructure mapping (DNS, WHOIS, BGP, SSL)
- Domain cluster analysis across impersonated brands
- Threat actor profiling and geo-attribution
- Actionable IoCs (IPs, domains, hashes, YARA rules)
- Defense recommendations prioritized by impact
